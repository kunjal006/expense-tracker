# 💰 Expense Tracker

A terminal-based expense tracking app built with Python.

## Features
- Add, View, Update, Delete expenses
- Category-wise filtering
- Search by title
- Monthly summary with percentages
- Highest & Lowest expense finder
- JSON-based persistent storage

## Tech Stack
Python, OOP, JSON, File Handling, DateTime

## How to Run
```bash
python main.py
```
